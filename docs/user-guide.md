## Managing Callings

### Calling Status Workflow

Callings move through these statuses:

1. **Pending**: Initial proposal, awaiting approval
2. **Stake Presidency Approved**: Approved by stake presidency, may need high council approval
3. **High Council Approved**: Approved by high council (for applicable positions)
4. **Called**: Calling has been extended and accepted by member
5. **LCR Updated**: Information has been updated in LCR system
6. **On Hold**: Temporarily paused for various reasons

### Approval Workflow

PENDING ‚Üí APPROVED (Stake Presidency) ‚Üí HC_APPROVED (High Council) ‚Üí CALLED ‚Üí LCR_UPDATED

**Note**: Not all callings require High Council approval. Some positions can go directly from Stake Presidency Approved to Called.

### Creating a New Calling

1. Navigate to any callings list
2. Click **Add New**
3. Fill in required information:
   - **Unit**: Select the appropriate unit
   - **Organization**: Choose the organization
   - **Position**: Select the position to be filled
   - **Status**: Set initial status (usually Pending)
4. Optional information:
   - **Currently Serving**: Name of current person in position
   - **Proposed Replacement**: Name of proposed person
   - **Dates**: Called, sustained, set apart dates
   - **Approvals**: Track presidency and high council approvals
   - **Notes**: Additional information

## Navigation

### Filtering Callings

Use the filter buttons to view specific calling subsets:
- **Active**: Shows all callings needing attention
- **All**: Shows every calling in the system

### Search Functionality

Use the search bar to find callings by:
- Member names
- Position titles
- Organization names
- Unit names
- Proposed replacement names

### List Views

**Calling List**: Traditional table view with all callings
**Callings by Unit**: Groups callings by unit for easier organization management

## Understanding Status Badges

Each calling displays a colored status badge:
- **Blue (Warning)**: Initial statuses (Pending)
- **Green (Success)**: Progress statuses (Stake Presidency Approved, High Council Approved, Called)
- **Yellow (Warning)**: Caution statuses (On Hold)
- **Gray (Info)**: Administrative statuses (LCR Updated)

## Tips for Effective Use

### For Bishoprics and Branch Presidencies
- Use the **Active** filter to focus on callings needing attention
- Check the dashboard regularly for quick overview
- Use the **Callings by Unit** view for organized reviews

### For Stake Presidencies
- Filter by specific units when reviewing
- Use search to quickly find specific positions or members
- Monitor the status chart for overall progress tracking

### General Best Practices
- Update calling status regularly to maintain accuracy
- Use notes field for important information
- Keep dates current for proper tracking
- Mark callings as LCR Updated after system updates

## Common Workflows

### Extending a New Calling
1. Create calling with status "Pending"
2. Get stake presidency approval, update status to "Stake Presidency Approved"
3. If required, get high council approval, update status to "High Council Approved"
4. Extend calling to member, update status to "Called"
5. Update LCR system, mark as "LCR Updated"

### Replacing an Existing Calling
1. Note current serving member
2. Add proposed replacement
3. Follow standard workflow above
4. Archive or mark previous calling appropriately

### Managing Multiple Callings
- Use unit grouping for organizational reviews
- Filter by active status for focused work sessions
- Use search for quick lookups during meetings

## Troubleshooting

### Common Issues

**Cannot find a calling**: Use the search function or check if filters are applied

**Wrong page after editing**: The system should return you to your previous view - if not, use browser back button

**Status confusion**: Refer to the status workflow section above for clarification
